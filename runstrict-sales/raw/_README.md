# RunStrict Sales — Raw Sources

Drop new sources here as: `<source-id>/<original-filename>`

Where `<source-id>` follows: `<publisher-or-origin>-YYYY-MM-DD[-<short-slug>]`

Examples:
```
raw/runstrict-ga-2026-05/dashboard-export.csv
raw/appsumo-pricing-teardown-2026-04/article.html
raw/lenny-newsletter-2026-04-30-onboarding/article.md
raw/partner-call-gymchain-2026-05-10/notes.md
```

## Hard rule

These files are **immutable**. The agent reads them but never edits, renames,
or deletes them.
