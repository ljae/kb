# Education Consulting — Raw Sources

Drop new sources here as: `<source-id>/<original-filename>`

Where `<source-id>` follows: `<publisher-or-origin>-YYYY-MM-DD[-<short-slug>]`

Examples:
```
raw/client-meeting-2026-05-12-s2026-01/notes.md
raw/admissions-report-mit-2026/report.pdf
raw/parent-call-2026-04-08-s2026-03/transcript.txt
```

## PII rule

Files in this folder MAY contain real names. They never leave `raw/`. The
agent reads them but the wiki uses anonymized handles only. A handle map
lives at `_PRIVATE_handle-map.md` (gitignored).

## Hard rule

These files are **immutable**. The agent never edits, renames, or deletes them.
